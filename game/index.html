准备

一台LINUX服务器（其实用自己的win机也可以开，但我比较喜欢用阿里云的Linux云服弄这些东西

如果你买的是云服务，配置可以先用1C4G买，不够了再往上加即可，有钱也可先直接买4C8G

网络的话，我记得以前开发大概快10个人玩，我买的是4-6M的外网宽带了，1-2M大概只够4个人玩吧，看自己的需要也可以先买低在往上加

操作喜欢的话，我比较喜欢Centos，服务器用的Centos7.4，其他Linux安装方法类比即可

如果你也是阿里云服务器 记得在安全组-配置规则里面开放端口 MC默认的是25565 不怕攻击图方便也可以直接设置-1/-1全放开放:D

安装Screen（可选）

为什么要安装Screen :D ?

为了让服务器一直运行~，你shh断了服也关掉了，但是我笔记本其实很少关机…我的Xshell也没关过…所以其实对我来说按不按都一样，不过还是记录一下比较这个东西很好用，使用起来有点类似VNC吧，可以开很多窗口（会话）

# 安装Screen
yum install screen -y
# 启动 创建窗口
screen -S screen_name
# 退出窗口
ctrl+A+D
# 查看所有窗口（显示session id
screen -list
screen -ls
# 回到某个窗口
screen -r
screen -r session_id
screen -r screen_name
# 删除某个窗口
screen -X -S session quit
# 以上的命令基本够我玩了 其他还有可以问百度哦

简单操作下：

安装JAVA

Linux服务器准备好之后需要安装JAVA

可以使用yum安装，也可以用离线包安装，这里我用离线包安装

# 创建Java安装路径并上传离线包
mkdir -p /server/java

# 用文件上传工具上传完毕之后解压
tar -xzvf jdk-8u161-linux-x64.tar.gz 

# 编辑profile文件,这就相当于win机的添加path环境变量吧
vim /etc/profile

# 扣i 编辑里面的内容 在结尾处添加下面的内容
# set Java environment
JAVA_HOME=/server/java/jdk1.8.0_161
PATH=$JAVA_HOME/bin:$PATH
CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
export JAVA_HOME
export PATH
export CLASSPATH
# 完毕扣esc输入:wq保存

#编译配置文件
source /etc/profile

#这时候Java已经装完啦 java -v javac 可以验证 下面可以跑服务器了

#如果重启之后java又找不到了重新 source /etc/profile 一下就好了 具体为啥的话可以自己度娘解决方法很多
#我是修改 ~/.bash_profile 
vim ~/.bash_profile
#在export PATH= 后添加$PATH:    

#也可以修改 ~/.bashrc 在其中添加环境变量或者添加source /etc/profile

下载server.jar

我自己是在官网上下的：官方下载

我记得官网上一直显示的是最新的服务器端jar包，2021.06.07我自己下载是 1.16.5的jar，后面如果更新了官网应该会显示下载最新的版本

emm，历史的版本得百度百度也都是可以找得到得，我记得有网站可以下载历史的jar，太久没弄找不到了，有空看到了再补上

启动

纯净启动也很简单，如果你也是从官网下载的jar就能看到

# 进入screen创建的mc窗口
screen -r Minecraft
# 执行
java -Xmx1024M -Xms1024M -jar minecraft_server.1.16.5.jar nogui

第一次启动会启动失败，需要你修改运行服务器目录下的eula.txt

修改为 eula=true

修改之后再次启动应该就能运行咯，但是！如果你是盗版侠，你即将被无情拒绝加入，应为你的服务器并没有关闭正版验证~

运行服务器之后ctrl+c停止服务器，你能看到你的目录下多了很多文件夹

通过名字不难理解这些文件是什么意思

world文件夹就是你刚刚生成的世界，不喜欢就删除掉 rm -rf world/ 再重新运行服务器就会生成新的世界，需要备份就直接copy保存这个文件夹也就行咯，你甚至可以把其他世界内容复制到里面去，替换你服务器的世界。

其他的嘛能用到再补充

修改server.properties

# server.properties是世界的相关配置设置文件，个人感觉必须要设置的有几个
# 关闭在线模式 允许盗版侠进入服务器
online-mode=false
# 运行飞行，不开启悬停在空中会被服务器自动踢出
allow-flight=true
# 剩下的难度 人数之类的看个人吧

详细的设置和解释可以参考：官方 server properties
